Hexo 使用日记

最近觉得学习效率略低，需要把自己学的东西整理一下，于是准备搭一个自己的博客网站，经大佬介绍，决定使用HEXO + Github Pages

HEXO是什么

  • Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

Github Pages 又是什么

  • 简单来说，Github Page 是github提供给广大用户的一个免费的个人网站，你可以用它做任何自己想做的事(不包括违法乱纪)
  • 用户域名：http:// [your github name] .github.io
  • 同时，你也可以把这个网站指向自己的域名(方法后面再说)

想详细了解? 戳这(Github Pages官网)

安装HEXO

安装前提





此处默认已经安装好 Node.js 和 Git ,没有安装的小伙伴自行安装

开始安装

  • 打开Git Bash，输入以下命令
    npm i -g hexo-cli
  • 等它运行完成，HEXO就已经安装在你的系统里了
  • 检查是否安装成功
    hexo -v

创建一个HEXO项目

  • 打开Git Bash,通过cd命令到你想去的目录，或打开资源管理器，点击右键

  • 直接在你想要的目录打开Git Bash

以下操作均需要在Git Bash进行，使用其他带有bash功能的命令行工具(如 Cmder)会出现很多问题

输入以下命令
hexo init [your dir] && cd [your dir]
npm i
