Nunjucks + Egg.js后端渲染页面

   最近开发的项目有一个需求，需要能够打印填写过后的最终成绩单，考虑过三种方案：到前端用vue画一个页面打印，导出成Word/PDF或者后台输出HTML，显然最后一种是最方便的。

于是乎去翻了Egg的文档，有一个插件叫egg-view-nunjucks，是基于Egg自带的egg-view和后端渲染引擎Nunjucks的后端渲染插件

egg-view

=> egg-view 文档
egg-view是Egg.js内置的模板解决方案，开发者可以利用egg-view开发模板插件也可以用它和已有模板插件配合使用。

Nunjucks

Nunjucks是一个后端模板引擎，可以把后端请求的数据渲染到写好的模板上返回html文档。
